### Getting Started [Bash]
1. Clone the repo
```sh
git@github.com:ariopri/MacroProject-Hiking.git
```
3. Go to frontend
```sh
cd frontend/
```
4. Install YARN packages
```sh
yarn
```
5. Starting
```JS
yarn run dev
```
